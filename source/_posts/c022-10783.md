## 內容
給定兩數 a, b (0 <= a <= b <= 100)，找出 a 與 b 之間所有奇數的和。

第一列有一個整數 T (1 <= T <= 100)代表測資數。
## 範例輸入
2
1
5
3
5
## 範例輸出
Case 1: 9
Case 2: 8
## 想法
從 a 遍歷到 b ，若是奇數則計算。

{% collapse 程式碼 %}
```cpp
#include <bits/stdc++.h>
#define fastio ios_base::sync_with_stdio(false), cin.tie(nullptr)
using namespace std;

int main(){
    fastio;
    int T, a, b, sum;
    cin >> T;
    for(int c = 1; c <= T; ++c){
        sum = 0;
        cin >> a >> b;
        for(int i = a; i <= b; ++i){
            if(i & 1) sum += i;
        }
        cout << "Case " << c << ": " << sum << "\n";
    }
    return 0;
}
```
