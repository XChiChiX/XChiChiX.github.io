a002: 簡易加法
<!--more-->
題目連結:https://zerojudge.tw/ShowProblem?problemid=a002
## 內容
請寫一個程式，讀入兩個數字，並求出它們的和。
## 範例輸入
5 10
1 2
## 範例輸出
15
3
## 想法
基本輸入輸出。

{% collapse 程式碼 %}
```cpp
#include <iostream>
using namespace std;

int main(){
    int a, b;
    cin >> a >> b;
    cout<< a + b;
    return 0;
}
```
