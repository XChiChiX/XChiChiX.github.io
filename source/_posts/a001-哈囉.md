## 內容
學習所有程式語言的第一個練習題 
請寫一個程式，可以讀入指定的字串，並且輸出指定的字串。

比如：輸入字串 "world", 則請輸出 "hello, world"
## 範例輸入
world
C++
hello, C++
## 範例輸出
hello, world
hello, C++
hello, Taiwan
## 想法
基本輸入輸出。

{% collapse 程式碼 %}
```cpp
#include <iostream>
using namespace std;

int main(){
    string a;
    cin >> a;
    cout << "hello, " << a;
    return 0;
}
```
