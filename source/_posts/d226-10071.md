## 內容
某一個粒子有一初速度和等加速度。假設在 t 秒後此粒子的速度為 v ，請問這個粒子在 2t 秒後所經過的位移是多少。

每組測資一列，分別代表 v (-100 <= v <= 100)、t (0 <= t <= 200)。
## 範例輸入
0 0
5 12
## 範例輸出
0
120

#include <bits/stdc++.h>
#define fastio ios_base::sync_with_stdio(false), cin.tie(nullptr)
using namespace std;

int main(){
    fastio;
    int t, v;
    while(cin >> t){
        cin >> v;
        cout << 2 * t * v << "\n";
    }
}
