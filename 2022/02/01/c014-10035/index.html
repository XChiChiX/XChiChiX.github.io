CPE 一顆星選集 49 道必考題

題目連結:https://zerojudge.tw/ShowProblem?problemid=c014

內容

給定兩個正整數，進行加法運算，輸出產生幾次進位。

最後一列有 2 個 0 代表輸入結束。

範例輸入

123 456
555 555
123 594
0 0

範例輸出

No carry operation.
3 carry operations.
1 carry operation.

想法

使用 string 分別存取兩數，開始模擬直式運算(記得考慮各種可能的數字，我卡超久= =)。
